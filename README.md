# 📱 App Store Project

A modern, responsive **App Store UI** built with **React + Vite + TailwindCSS**.  
This project allows users to install/uninstall apps, persist data in **localStorage**, and visualize ratings using **Recharts**.

---

## 🚀 Features
- 🔍 Browse and install apps
- 🗑️ Uninstall apps (based on `id`)
- 💾 Persistent storage using `localStorage`
- 🎨 Beautiful UI with TailwindCSS
- 📊 Interactive charts with Recharts (PieChart for ratings)
- ⚡ Fast development with Vite
- 🖼️ Error/Empty state pages with custom design

---

## 🛠️ Tech Stack
- **React** (Frontend framework)
- **Vite** (Bundler & Dev server)
- **TailwindCSS** (Styling)
- **Recharts** (Data visualization)
